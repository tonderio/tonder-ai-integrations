# Example Prompts

Use these prompts to test the Tonder Web SDK integration skill.

## HTML card payment

```text
Use the Tonder Web SDK integrator to add embedded card payments to this static HTML checkout page.
```

## React card payment

```text
Use the Tonder Web SDK integrator to add embedded card payments to this React checkout component.
```

## Angular card payment

```text
Use the Tonder Web SDK integrator to add redirect card payments to this Angular checkout page.
```

## Saved cards

```text
Use the Tonder Web SDK integrator to add saved-card payments to this checkout. Ask me where to obtain the secure token if it is not already present.
```

## Payment methods

```text
Use the Tonder Web SDK integrator to add OXXO Pay and SPEI payment options to this checkout.
```

## SafetyPay banks

```text
Use the Tonder Web SDK integrator to add SafetyPay bank payments using getPaymentMethodBanks().
```
