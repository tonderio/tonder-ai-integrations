# Release 0.1.2

## Summary

Patch release for GitHub marketplace installs of the Tonder Web SDK plugin.

## Fixed

- Claude Desktop and Claude Code installs from the GitHub marketplace now include the compiled `tonder-docs` MCP server at `mcp/dist/server.js`.
- The MCP server is bundled into a self-contained file and no longer depends on plugin-local `node_modules`.

## Verification

- `npm test`
- `npm run build`
- `claude plugin validate ./plugins/claude-code/tonder-web-sdk`
- `claude plugin validate .`
- Codex plugin validation
- Clean MCP smoke test without `node_modules`
