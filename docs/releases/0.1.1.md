# Release 0.1.1

## Summary

Initial public release candidate for the Tonder AI Integrations repository with the `tonder-web-sdk` plugin for Claude Code and Codex.

## Included

- Claude Code plugin marketplace package.
- Codex plugin marketplace package.
- Bundled local `tonder-docs` MCP server.
- Tonder Web SDK integration skill for HTML, React, and Angular projects.
- Maintainer documentation, changelog, MIT license, and security policy.

## Verification

Before publishing, maintainers must run:

```bash
claude plugin validate ./plugins/claude-code/tonder-web-sdk
claude plugin validate .
python3 /Users/dave/.codex/skills/.system/plugin-creator/scripts/validate_plugin.py ./plugins/codex/tonder-web-sdk
cd packages/tonder-mcp && npm test && npm run build
```

## Install smoke tests

Claude Code:

```bash
claude plugin marketplace add tonderio/tonder-ai-integrations
claude plugin install tonder-web-sdk@tonder-ai-integrations
```

Codex:

```bash
codex plugin marketplace add tonderio/tonder-ai-integrations --ref main
codex plugin add tonder-web-sdk@tonder-ai-integrations
```
