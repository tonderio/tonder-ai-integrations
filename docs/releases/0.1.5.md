# Release 0.1.5

## Summary

Patch release for secure-field container layout guidance.

## Fixed

- HTML, React, and Angular recipes now include the recommended container CSS:

  ```css
  .card-field {
    width: 100%;
    min-height: 90px;
  }
  ```

- The plugin now explicitly tells agents not to pass container sizing such as `height` or `min-height` through `customization.card_fields.styles`; SDK styles are for secure input, label, error, and icon styling inside the iframe.
